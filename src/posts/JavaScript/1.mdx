## 문제 상황

스크롤하거나 입력할 때, 이벤트가 너무 자주 발생하면  
성능 저하, 과도한 렌더링이 발생할 수 있습니다.

이를 막기 위해 `debounce`와 `throttle`을 사용합니다.

## debounce

- 마지막 이벤트 이후 일정 시간 동안 이벤트가 없을 때만 실행
- 주로 **검색창 입력**, **윈도우 리사이즈** 등에 사용

```ts
function debounce(fn, delay) {
  let timer;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), delay);
  };
}
