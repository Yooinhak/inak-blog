이 포스트는 App Router 기준입니다.

## OG 태그란?

트위터, 카카오톡, 슬랙 등에 URL을 공유하면 예쁘게 카드가 뜨죠?  
이걸 가능하게 해주는 것이 Open Graph 메타 태그입니다.

## 설정 위치

Next.js App Router에선 `metadata`를 통해 설정할 수 있습니다.

```ts
// app/page.tsx 또는 app/route.tsx
export const metadata = {
  title: '이낙 블로그',
  description: '프론트엔드 개발자 블로그',
  openGraph: {
    title: '이낙 블로그',
    description: 'Next.js, React 관련 개발 이야기',
    url: 'https://inak.dev',
    siteName: 'inak-blog',
    images: [
      {
        url: 'https://inak.dev/og-image.png',
        width: 1200,
        height: 630,
      },
    ],
    locale: 'ko_KR',
    type: 'website',
  },
};
